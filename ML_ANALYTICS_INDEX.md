## 🔧 Customization Guide

### Change Accent Color
```css
/* Edit in ml_analytics_premium.html <style> section */
:root {
    --ai-purple: #your-color;  /* Change primary */
    --ai-blue: #your-color;    /* Change secondary */
}
```

### Add New KPI Card
```html
<div class="kpi-card glass">
    <div class="kpi-header">
        <div class="kpi-icon">
            <i class="fas fa-your-icon"></i>
        </div>
    </div>
    <div class="kpi-label">Your Metric</div>
    <div class="kpi-value">1,234</div>
    <div class="kpi-trend">
        <span class="trend-badge trend-up">
            <i class="fas fa-arrow-up"></i> Growing
        </span>
    </div>
</div>
```

### Create Custom Chart
```javascript
const myChart = new Chart(ctx, {
    type: 'line',  // or: bar, pie, doughnut, radar
    data: {
        labels: ['Q1', 'Q2', 'Q3', 'Q4'],
        datasets: [{
            label: 'Revenue',
            data: [12, 19, 3, 5],
            borderColor: '#a855f7',
            backgroundColor: 'rgba(168, 85, 247, 0.1)',
            borderWidth: 3,
            tension: 0.4
        }]
    }
});
```

## 🆘 Support & Troubleshooting

### Common Issues

**Q: Dashboard not loading?**
```
A: Check console (F12) for errors
   Ensure internet connection (CDN dependencies)
   Try different browser (Chrome/Edge recommended)
```

**Q: Charts not displaying?**
```
A: Verify Chart.js loaded: console.log(typeof Chart)
   Clear browser cache
   Reload page (Ctrl+F5)
```

**Q: File upload fails?**
```
A: Check file format (.xlsx, .xls, .csv)
   Verify file size (< 10MB recommended)
   Ensure columns are properly formatted
```

**Q: Blur effects missing?**
```
A: Check browser support for backdrop-filter
   Safari: Enable in Experimental Features
   Firefox: Set layout.css.backdrop-filter.enabled = true
```
